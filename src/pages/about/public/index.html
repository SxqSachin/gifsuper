关于Super Gif

这是一个帮助您在线编辑Gif的工具站。

即开即用无需登陆等多余操作，减少您的使用成本。

Gif编辑过程中不包含任何网络请求，操作均在您的终端设备上直接计算。

目前包含下列功能：

  1. Gif调速
  2. Gif倒放
  3. 开启/关闭Gif循环
  4. 添加文字
  5. 复制/删除Gif帧
  6. 抽取半数帧
  7. Gif裁剪（范围裁剪，范围删除）

正在着手开发的功能：

  1. 大尺寸Gif显示优化
  2. Gif压缩
  3. Gif尺寸裁剪
  4. Gif拼接
  5. Gif分离（分解为多张Gif）

功能会不断扩充，期待您的建议与反馈，帮助Gif Super变得更好。
